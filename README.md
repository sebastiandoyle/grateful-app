# Grateful

A gratitude journal that earns its place on your home screen. Write three good things each day, get weekly recaps that actually make you feel something, and build the simplest habit with the highest ROI in positive psychology.

## Features

- **Daily gratitude entries** with guided prompts
- **Weekly recaps** — AI-summarized themes from your entries
- **Streak tracking** to build the journaling habit
- **Photo attachments** for visual gratitude moments
- **Mood correlation** — see how gratitude practice maps to your mood over time
- **Privacy-first** — all data stays on device

## Tech Stack

- SwiftUI
- SwiftData
- PhotosUI
- Charts framework

## Getting Started

```bash
git clone https://github.com/sebastiandoyle/grateful-app.git
cd grateful-app
open *.xcodeproj
```

Requires Xcode 15+ and iOS 17+.

## Why Another Gratitude App?

Most gratitude apps are either too simple (just a text field) or too complex (gamification, social features, upsells). Grateful sits in the middle: guided enough to keep you writing, minimal enough to not become another chore.
